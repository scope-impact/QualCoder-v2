TextMiningEngine: word frequency analysis, collocation detection, exact/regex text search across coded segments.

## Acceptance Criteria
<!-- AC:BEGIN -->
### TextMiningEngine Interface
- [ ] #1 `calculate_word_frequencies(sources, config) -> List[WordFrequency]`
- [ ] #2 `find_collocations(sources, config, window_size) -> List[Collocation]`
- [ ] #3 `search_exact(sources, query) -> List[SearchResult]`
- [ ] #4 `search_regex(sources, pattern) -> List[SearchResult]`

### Configuration
- [ ] #5 `TextMiningConfig`: min_word_length, stopwords, case_sensitive, stemming
- [ ] #6 Support multiple languages for stopwords

### Word Frequency Analysis
- [ ] #7 Tokenizes text, filters stopwords, counts frequencies
- [ ] #8 Returns sorted by frequency descending
- [ ] #9 Includes source distribution per word

### Collocation Detection
- [ ] #10 Finds frequently co-occurring word pairs
- [ ] #11 Calculates PMI (Pointwise Mutual Information) score
- [ ] #12 Configurable window size for adjacency

### Search
- [ ] #13 Exact search: case-sensitive/insensitive option
- [ ] #14 Regex search: validates pattern before execution
- [ ] #15 Returns context (surrounding text) with matches

### Performance
- [ ] #16 Streaming processing for large source collections
- [ ] #17 Progress callback support
