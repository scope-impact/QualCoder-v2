## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Exporters: Excel (XLSX with formatting), PDF (for graphs), Image (PNG for charts). Support all report types.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
### Exporter Interfaces
- [ ] #1 `ExcelExporter.export(report, path) -> Result[Path, ExportError]`
- [ ] #2 `PDFExporter.export(graph, path) -> Result[Path, ExportError]`
- [ ] #3 `ImageExporter.export(chart, path, format) -> Result[Path, ExportError]`

### Excel Export (XLSX)
- [ ] #4 Support all report types: Frequency, CoOccurrence, CoderComparison
- [ ] #5 Formatted headers, column widths, number formats
- [ ] #6 Multiple sheets for complex reports

### PDF Export
- [ ] #7 Network graphs rendered to PDF
- [ ] #8 Configurable page size and orientation
- [ ] #9 Title and legend included

### Image Export
- [ ] #10 PNG format for charts and graphs
- [ ] #11 Configurable resolution (DPI)
- [ ] #12 Transparent background option

### CSV Export
- [ ] #13 Simple tabular export for data interchange
- [ ] #14 Configurable delimiter

### Error Handling
- [ ] #15 `ExportError` union: FileNotWritable, DiskFull, RenderingFailed
- [ ] #16 Returns Result[Path, ExportError]
<!-- AC:END -->
