## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Bridge between domain events and UI signals for the Case context. Enable UI observation of case changes from both UI and Agent. Thread-safe emission.

## Implementation Notes

**For PyQt6:**
- Inherits from `QObject` via BaseSignalBridge
- Signals declared as `pyqtSignal(object)`
- Uses `QMetaObject.invokeMethod` with `Qt.QueuedConnection` (via base class)
