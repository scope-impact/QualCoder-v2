## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
NetworkGraphGenerator: build code relationship graphs, calculate node positions, support various layout algorithms.

## Acceptance Criteria
<!-- AC:BEGIN -->
### NetworkGraphGenerator Interface
- [ ] #1 `generate_code_graph(codes, categories, config) -> Graph`
- [ ] #2 `generate_cooccurrence_graph(matrix, threshold) -> Graph`
- [ ] #3 `calculate_layout(graph, layout_type) -> GraphLayout`

### Layout Algorithms
- [ ] #4 `LayoutType.TREE`: hierarchical category layout
- [ ] #5 `LayoutType.FORCE`: force-directed spring layout
- [ ] #6 `LayoutType.CIRCULAR`: codes arranged in circle

### Graph Building
- [ ] #7 Nodes from codes and categories
- [ ] #8 Edges from parent-child relationships
- [ ] #9 Co-occurrence edges from matrix (weighted)

### Pure Functions
- [ ] #10 `apply_force_layout(graph, iterations) -> List[NodePosition]` - pure calculation
- [ ] #11 `apply_tree_layout(graph, root_id) -> List[NodePosition]`
- [ ] #12 `prune_graph(graph, min_weight) -> Graph` - remove weak edges

### Export
- [ ] #13 Graph exportable to JSON for visualization library
- [ ] #14 Position data included for rendering

### Configuration
- [ ] #15 `GraphConfig`: max_nodes, min_edge_weight, layout_params
