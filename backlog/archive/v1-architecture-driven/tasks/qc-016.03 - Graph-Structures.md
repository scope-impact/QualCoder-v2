## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Graph domain types: GraphNode, GraphEdge, GraphConfiguration, GraphLayout. Support network visualization and code relationship mapping.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
### Graph Value Objects (frozen dataclasses)
- [ ] #1 `GraphNode`: node_id, label, color, node_type (CODE|CATEGORY|FILE|CASE), metadata
- [ ] #2 `GraphEdge`: source_id, target_id, weight, edge_type (CONTAINS|OVERLAPS|LINKED)
- [ ] #3 `GraphLayout`: nodes (List[NodePosition]), layout_type (TREE|FORCE|CIRCULAR)
- [ ] #4 `NodePosition`: node_id, x, y

### GraphConfiguration (frozen dataclass)
- [ ] #5 `GraphConfiguration`: root_nodes, max_depth, include_types, layout_type
- [ ] #6 Immutable - changes create new configuration

### Pure Functions
- [ ] #7 `build_graph(codes, categories, config) -> Graph` - creates graph structure
- [ ] #8 `calculate_positions(graph, layout_type) -> GraphLayout` - layout algorithm
- [ ] #9 `filter_graph(graph, filter) -> Graph` - prune nodes/edges

### Graph Algorithms (Pure)
- [ ] #10 `find_connected_components(graph) -> List[List[NodeId]]`
- [ ] #11 `calculate_centrality(graph) -> Dict[NodeId, float]`
- [ ] #12 `detect_cycles(graph) -> List[List[NodeId]]`
<!-- AC:END -->
