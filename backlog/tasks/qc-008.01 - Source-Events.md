## Acceptance Criteria
<!-- AC:BEGIN -->
### Source Lifecycle Events (frozen dataclasses extending DomainEvent)
- [ ] #1 `SourceRenamed` event: source_id, old_name, new_name
- [ ] #2 `SourceMemoUpdated` event: source_id, old_memo, new_memo
- [ ] #3 `SourceDeleted` event: source_id, source_type, had_segments (bool)
- [ ] #4 `SourceExported` event: source_id, export_path, format

### Transcript Events
- [ ] #5 `SourceTranscribed` event: source_id, transcript_length, language (optional)
- [ ] #6 `SpeakersDetected` event: source_id, speakers (list of speaker names), pattern_format

### Cross-Context Events (Published Language)
- [ ] #7 `SourceCaseLinked` event: source_id, case_id, position (optional for partial)
- [ ] #8 `SourceCaseUnlinked` event: source_id, case_id

### Survey Events
- [ ] #9 `SurveyImported` event: source_ids (list), row_count, column_count, attribute_types_created

### Event Structure
- [ ] #10 All events follow standard structure: event_id, event_type, aggregate_id, occurred_at, caused_by, payload
- [ ] #11 Events registered in Source context event catalog
- [ ] #12 Cross-context events (SourceCaseLinked) are Published Language for Case context
<!-- AC:END -->
