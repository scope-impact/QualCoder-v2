## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
TrustLevelController: get/set global trust level, per-tool trust overrides, rate limiting configuration, pause/resume agent operations.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
### TrustLevel Enum (from shared domain)
- [ ] #1 `TrustLevel`: AUTONOMOUS (no approval), NOTIFY (execute + notify), SUGGEST (approval required), REQUIRE (explicit approval)

### TrustLevelController Interface
- [ ] #2 `get_global_trust_level() -> TrustLevel`
- [ ] #3 `set_global_trust_level(level: TrustLevel) -> Result[None, Error]`
- [ ] #4 `get_tool_trust_level(tool_name) -> TrustLevel` - per-tool override
- [ ] #5 `set_tool_trust_level(tool_name, level) -> Result[None, Error]`
- [ ] #6 `reset_to_defaults() -> None`

### Rate Limiting
- [ ] #7 `get_rate_limit() -> ActionsPerMinute`
- [ ] #8 `set_rate_limit(actions_per_minute: int) -> Result[None, Error]`
- [ ] #9 `is_rate_limited() -> bool` - check current state

### Agent Control
- [ ] #10 `pause_agent() -> Result[None, Error]` - temporarily stop processing
- [ ] #11 `resume_agent() -> Result[None, Error]`
- [ ] #12 `is_agent_paused() -> bool`

### Persistence
- [ ] #13 Trust levels persisted to project settings
- [ ] #14 Per-tool overrides stored separately

### Events
- [ ] #15 `TrustLevelChanged` event: old_level, new_level, tool_name (optional)
<!-- AC:END -->
