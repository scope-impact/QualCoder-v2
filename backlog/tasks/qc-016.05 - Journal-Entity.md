## Description

Define domain entity for Research Journals (reflexive research diaries). Journals document researcher thinking, decisions, and reflections throughout analysis.

## Original QualCoder Reference

- Module: `journals.py`
- Table: `journal` (jid, name, jentry, date, owner)

## Acceptance Criteria

### Entity: Journal
- [ ] #1 `id: JournalId`
- [ ] #2 `name: str` - Journal title (unique per project)
- [ ] #3 `content: str` - Rich text content (markdown supported)
- [ ] #4 `owner: str` - Coder who created it
- [ ] #5 `created_date: datetime`
- [ ] #6 `modified_date: datetime`

### Value Objects
- [ ] #7 `JournalId`
- [ ] #8 `JournalContent` (validated, preserves markdown)

### Domain Events
- [ ] #9 `JournalCreated(journal_id, name, owner)`
- [ ] #10 `JournalUpdated(journal_id, modified_date)`
- [ ] #11 `JournalDeleted(journal_id)`
- [ ] #12 `JournalExported(journal_id, format, path)`

### Validation
- [ ] #13 Name required, unique within project
- [ ] #14 Owner required
