## Acceptance Criteria
<!-- AC:BEGIN -->
### Coding Operations
- [ ] #1 `Q` - Quick mark selection with selected code
- [ ] #2 `U` - Unmark code at cursor position
- [ ] #3 `N` - Open new code dialog
- [ ] #4 `V` - In-vivo code (create code from selection)
- [ ] #5 `M` - Add/edit memo for segment at cursor
- [ ] #6 `A` - Annotate selection
- [ ] #7 `R` - Show recent codes menu

- [ ] #10 `Ctrl+F` - Focus search box
- [ ] #11 `Ctrl+1` - Go to next file
- [ ] #12 `Ctrl+2` - Go to latest coded file
- [ ] #13 `O` - Cycle through overlapping codes at cursor
- [ ] #14 `B` - Set bookmark at cursor

### Code Boundary Adjustment
- [ ] #15 `Shift+Left` - Extend code boundary left
