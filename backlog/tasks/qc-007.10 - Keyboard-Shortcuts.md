## Acceptance Criteria
<!-- AC:BEGIN -->
### Coding Operations
- [x] #1 `Q` - Quick mark selection with selected code
- [x] #2 `U` - Unmark code at cursor position
- [x] #3 `N` - Open new code dialog (emits signal)
- [x] #4 `V` - In-vivo code (create code from selection)
- [x] #5 `M` - Add/edit memo for segment at cursor (handler ready)

- [x] #7 `R` - Show recent codes menu
- [x] #8 `I` - Toggle important flag on segment (handler ready)
- [x] #9 `Ctrl+Z` - Undo last unmark
