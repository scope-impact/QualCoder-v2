## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Implement visualization components for the design system based on ADR-001 library decisions. These components wrap external libraries (PyQtGraph, NetworkX, wordcloud) with consistent design system styling and theming support.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
- [x] #1 ChartWidget - PyQtGraph wrapper with design system theming (bar, line, scatter, pie)
- [x] #2 NetworkGraphWidget - NetworkX + QGraphicsScene with interactive nodes/edges
- [x] #3 WordCloudWidget - wordcloud library wrapper with color scheme support
- [x] #4 PDFPageViewer - PyMuPDF wrapper with text selection overlay
- [x] #5 ImageAnnotationLayer - QGraphicsScene-based rectangle/polygon annotation
- [x] #6 HeatMapCell - DataTable cell variant with color intensity
- [x] #7 RelevanceScoreBar - Horizontal progress variant for search scores
- [x] #8 All components support dark/light theme switching
- [x] #9 Unit tests for each component
<!-- AC:END -->

## Dependencies to Add

```bash
uv add pyqtgraph networkx wordcloud numpy matplotlib pymupdf
```
