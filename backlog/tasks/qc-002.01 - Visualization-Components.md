## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Implement visualization components for the design system based on ADR-001 library decisions. These components wrap external libraries with consistent design system styling and theming support.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
- [ ] #1 ChartWidget - Statistical chart component (bar, line, scatter, pie)
- [ ] #2 NetworkGraphWidget - Interactive network graph with nodes/edges
- [ ] #3 WordCloudWidget - Word frequency visualization with color scheme support
- [ ] #4 PDFPageViewer - PDF page rendering with text selection overlay
- [ ] #5 ImageAnnotationLayer - Rectangle/polygon annotation on images
- [ ] #6 HeatMapCell - Data table cell variant with color intensity
- [ ] #7 RelevanceScoreBar - Horizontal progress variant for search scores
- [ ] #8 All components support dark/light theme switching
- [ ] #9 Unit tests for each component
<!-- AC:END -->

## Implementation Notes

**For PyQt6:**
- ChartWidget uses PyQtGraph for performance
- NetworkGraphWidget uses NetworkX with QGraphicsScene
- WordCloudWidget wraps the wordcloud library
- PDFPageViewer uses Qt PDF module
- ImageAnnotationLayer built on QGraphicsScene

## Dependencies to Add

```bash
uv add pyqtgraph networkx wordcloud numpy matplotlib
```
