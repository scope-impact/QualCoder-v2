## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Text mining value objects: WordFrequency, Collocation, SearchResult, StopWordList. Support word cloud and text analysis features.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
### Text Mining Value Objects (frozen dataclasses)
- [ ] #1 `WordFrequency`: word (str), count (int), sources (List[SourceId])
- [ ] #2 `Collocation`: words (Tuple[str, str]), frequency, pmi_score
- [ ] #3 `SearchResult`: source_id, position (TextPosition), context (str), match (str)
- [ ] #4 `StopWordList`: words (FrozenSet[str]), language

### TextMiningConfig (frozen dataclass)
- [ ] #5 `TextMiningConfig`: min_word_length, stopwords, case_sensitive, stemming
- [ ] #6 Immutable configuration for text analysis

### Pure Functions
- [ ] #7 `tokenize(text, config) -> List[str]` - pure tokenization
- [ ] #8 `calculate_word_frequencies(tokens) -> List[WordFrequency]`
- [ ] #9 `find_collocations(tokens, window_size) -> List[Collocation]`
- [ ] #10 `search_exact(text, pattern) -> List[SearchResult]`
- [ ] #11 `search_regex(text, pattern) -> List[SearchResult]`

### Invariants
- [ ] #12 `is_valid_regex(pattern) -> bool`
- [ ] #13 `is_valid_word(word, min_length) -> bool`
