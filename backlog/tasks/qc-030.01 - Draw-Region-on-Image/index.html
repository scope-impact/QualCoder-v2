    030.01 Draw Region

    Description¶

    As a Researcher, I want to draw regions on images so that I can mark areas for coding.

    Acceptance Criteria¶

    • [ ] #1 I can draw rectangular regions
    • [ ] #2 I can draw freeform regions
    • [ ] #3 I can resize and move regions
    • [ ] #4 Region boundaries are clearly visible
