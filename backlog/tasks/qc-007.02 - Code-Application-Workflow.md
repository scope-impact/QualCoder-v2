## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Implement the core workflow for applying codes to text selections. Wire signals between text selection and code application. Update highlights and code counts on application.

## Acceptance Criteria
<!-- AC:BEGIN -->
- [ ] #1 Apply selected code to text selection via Q key (quick mark)
- [ ] #2 In-vivo coding: create new code from selected text (V key)
- [ ] #3 Apply code from recent codes submenu (R key)
- [ ] #4 Remove code at cursor position (U key - unmark)
- [ ] #5 Undo last unmark operation (Ctrl+Z)
- [ ] #6 Visual feedback (flash/highlight) on code application
- [ ] #7 Code counts update in tree after apply/remove
- [ ] #8 `text_selected` signal enables code application buttons
- [ ] #9 `code_selected` signal sets active code for quick-mark
- [ ] #10 `code_applied` signal triggers highlight update
<!-- AC:END -->
