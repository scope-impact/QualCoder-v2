A modern, scholarly interface design for qualitative researchers. Featuring AI-powered coding assistance, streamlined workflows, and beautiful visualizations. Built with accessibility, warmth, and efficiency in mind.

Dashboard

Project overview with quick actions, recent activity, coding progress, and AI-powered suggestions at a glance.

Sources Manager

Modern file management with grid/list views, drag-and-drop folders, type filters, and bulk operations.

Coding Workspace

Three-panel coding interface with code tree, text editor with highlighting, and contextual details panel.

Case Manager

Participant and case management with attributes, linked sources, memos, and detailed case views.

Analysis & Insights

Data visualizations, code frequency charts, co-occurrence matrices, themes, and AI-generated insights.
