# Getting Started

This guide walks you through installing and using the QualCoder Design System in your PySide6 applications.

## Overview

The QualCoder Design System provides:

- **Design Tokens** — Consistent values for colors, spacing, typography
- **Core Components** — Buttons, inputs, cards, labels, badges
- **Layout Components** — App containers, panels, sidebars, toolbars
- **Form Components** — Search boxes, selects, textareas, color pickers
- **Data Components** — Tables, trees, lists, charts
- **Feedback Components** — Spinners, toasts, progress bars
- **Advanced Components** — Modals, context menus, editors
