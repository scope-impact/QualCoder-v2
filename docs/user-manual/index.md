# User Manual

Welcome to QualCoder v2 - a qualitative data analysis (QDA) application for coding and analyzing text, images, audio, and video.

!!! info "AI-Agent Ready"
    QualCoder v2 supports both human researchers and AI agents working together. All features are accessible via the UI and through MCP tools.

## What is QualCoder?

QualCoder is a free, open-source qualitative data analysis tool that helps researchers:

- **Code** text, images, audio, and video data
- **Organize** codes into categories and hierarchies
- **Analyze** patterns and themes in your data
- **Collaborate** with AI agents for assisted coding
